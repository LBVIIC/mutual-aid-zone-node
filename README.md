## 环境变量

|     变量名     |            描述            |
| :------------: | :------------------------: |
|   SECURT_KEY   |         Token Key          |
|  PASSWORD_KEY  |        MD5 加密 Key        |
| EMAIL_USERNAME |         发件邮箱名         |
| EMAIL_PASSWORD |        发件邮箱密码        |
|  DB_USERNAME   | MongoDB 用户名（生产环境） |
|  DB_PASSWORD   |  MongoDB 密码（生产环境）  |
